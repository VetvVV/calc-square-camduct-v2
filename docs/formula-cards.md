# Карточки формул изделий — Calc Square / Miro

Файл хранит рабочие карточки формул для расчёта площади развёртки.

Назначение файла:

- быстро дать руководителю понятные формулы расчёта расхода материала;
- хранить логику отдельно от UI;
- позже подключить эти данные в админский блок калькулятора: `Admin + ⚙ ON → Формулы / инженерная справка`.

## 1. Основные определения

```text
Sчистовая — площадь геометрии изделия без припусков, замков, соединителей и технологических добавок.
Sполная — площадь развёртки с припусками, замками, соединителями и прочими технологическими добавками.
```

Для руководителя и расчёта расхода материала главным показателем является:

```text
Sполная
```

Если `Sполная` пока не выведена, в карточке указывается `требует уточнения`, а `Sчистовая` остаётся как базовая геометрия.

Массу в этих карточках не считаем.
Количество не учитываем: каждая карточка описывает расчёт для 1 изделия.

## 2. Статусы

```text
Подтверждена — формула подтверждена проектной логикой / CAMduct / проверочным примером.
Выведена предположительно — формула логически выведена, но нужна сверка с CAMduct.
Временная — допускается как placeholder до уточнения.
Требует уточнения — формулы пока недостаточно для расчёта расхода материала.
```

## 3. Шифры изделий

```text
KRG — круглые изделия
PRM — прямоугольные изделия
KMB — комбинированные изделия
```

Старый код `R-001` считается историческим кодом для `KRG-001`.

# KRG-001 — Воздуховод круглый / труба прямошовная

**Категория:** круглые изделия  
**Старый код:** `R-001`  
**Технический ключ старого атласа:** `round-duct`

## Параметры

```text
D — диаметр, мм
L — длина, мм
t — толщина металла, мм
```

## Sчистовая

```text
Sчистовая = π × D × L / 1 000 000
```

## Sполная

```text
Sполная = (π × D + S1) × L / 1 000 000
```

где:

```text
S1 = S1FA + S1MA
```

Русский замок S1:

```text
t = 0.5 или 0.7 мм → S1 = 12.5 + 12.5 = 25 мм
t = 0.9 мм         → S1 = 14 + 14 = 28 мм
```

Не округлять `12.5` до `12` при расчёте полной площади развёртки.

## Пример

```text
D = 250 мм
L = 1000 мм
t = 0.5 мм
S1 = 25 мм

Sчистовая = 3.1416 × 250 × 1000 / 1 000 000
Sчистовая = 0.785 м²

Sполная = (3.1416 × 250 + 25) × 1000 / 1 000 000
Sполная = 0.810 м²
```

## Статус

```text
Sчистовая — подтверждена
Sполная — подтверждена по текущей логике S1
```

## Источник

```text
AGENTS.md / проектная логика R-001 / правило S1 / подтверждение владельца
```

## Не входит

```text
отверстия
tорцевые соединители
масса
раскладка на лист
отход материала
```

# KRG-002 — Отвод круглый / колено круглое

**Категория:** круглые изделия  
**Технический ключ старого атласа:** `round-elbow`

## Параметры

```text
D / A — диаметр отвода, мм
C / α — фактический угол производства, °
N / n — количество сегментов
R / RB — внутренний радиус / шейка, мм
L1, L2 — нижнее и верхнее удлинения стаканов, мм
C1 = C1F + C1M — припуск соединителя, мм
S1 = S1FA + S1MA — припуск шва, мм
```

## Sчистовая

```text
Методичка содержит формулу Sчистовая, но запись повреждена и требует восстановления скобок.
Sчистовая восстановлена по геометрии развёртки Yш/Yз без C1 и S1.

α = (C / (N − 1)) × π / 180
Yш = 2 × R × sin(α / 2) × kш
Yз = 2 × (R + Dmin) × sin(α / 2) × kз
```

## Sполная

```text
Sполная = Lразв × (π × Dmin + S1) / 1 000 000

Lразв берётся из таблицы угла:
L1 + L2 + вклад Yш/Yз по сегментам + (N − 1) × C1

C1 = C1F + C1M
S1 = S1FA + S1MA
```

В Excel-базе есть отдельные листы для углов:

```text
Отвод 90°
Отвод 60°
Отвод 45°
Отвод 30°
Отвод 15°
```

## Статус

```text
Проверена — формула восстановлена по Excel-базе, сформированной на основе CAMduct.
Проверена по Excel-базе CAMduct, не является условной.
```

## Источник

```text
Методичка основная 30.09.2021:
раздел “Отводы сегментные круглого сечения (колено круглое)”.

БазаОтвод_90_60_45_30_15_Киев_v2.0.0.xlsx:
листы “Отвод 90°”, “Отвод 60°”, “Отвод 45°”, “Отвод 30°”, “Отвод 15°”.
```

## Следующее действие

```text
Перенести формулу в расчётный engine и покрыть тестами.
```

# PRM-001 — Воздуховод прямоугольный

**Категория:** прямоугольные изделия
**Технический ключ старого атласа:** `rectangular-duct`

## Назначение

```text
PRM-001 — базовый прямоугольный воздуховод.
Это прямой прямоугольный участок без переходов, тройников, врезок и шумоглушителей.
Формула карточки описывает 1 изделие.
```

Не смешивать PRM-001 с прямоугольным переходом, тройником, врезкой или шумоглушителем.

## Параметры

```text
W — ширина, мм
H — высота, мм
L — длина, мм
s — толщина металла, мм
Z1/Z2 — технологические добавки / замки по развёртке старого калькулятора, мм
n — количество изделий
```

В старых калькуляторах эти параметры могли называться `A`, `B`, `L`:

```text
A → W
B → H
L → L
```

## Sчистовая

```text
Sчистовая = 2 × (W + H) × L / 1 000 000
```

`Sчистовая` — геометрическая площадь прямоугольного ствола без замков, припусков и торцевых соединений.

## Sполная

```text
Pполный = 2 × (W + Z1) + 2 × (H + Z2)
Sполная = Pполный × L / 1 000 000
```

Правила `Z1/Z2` по старому калькулятору:

```text
если L ≤ 200 мм:
Z1 = 0
Z2 = 15

если L > 200 мм и s ≤ 0.9 мм:
Z1 = 6
Z2 = 30

если L > 200 мм и s > 0.9 мм:
Z1 = 5
Z2 = 27
```

`L ≤ 200` — короткая деталь с отдельным правилом.  
Если есть количество `n`, итоговая площадь партии:

```text
Sпартии = S × n
```

Но карточка PRM-001 хранит формулу на 1 изделие.

## Пример

```text
W = 300 мм
H = 200 мм
L = 1000 мм
s = 0.7 мм

Z1 = 6 мм
Z2 = 30 мм

Sчистовая = 2 × (300 + 200) × 1000 / 1 000 000
Sчистовая = 1.000 м²

Pполный = 2 × (300 + 6) + 2 × (200 + 30)
Pполный = 1072 мм

Sполная = Pполный × L / 1 000 000
Sполная = 1072 × 1000 / 1 000 000
Sполная = 1.072 м²
```

## Статус

```text
Проверена — формула взята из старого рабочего калькулятора проекта.
Основной источник — внутренний проверенный калькулятор проекта, не СПВА.
Торцевые соединения C1/C2 не включать автоматически без подтверждения.
```

## Источник

```text
Старый калькулятор прямоугольных воздуховодов:
duct_calculator.html / Калькулятор_прямоугольных_воздуховодов.html

СПВА остаётся внешним справочным альтернативным источником для чистовой геометрии.
СПВА не заменяет основную формулу старого калькулятора с Z1/Z2.
```

## Открытые вопросы

```text
Подтвердить по CAMduct, совпадают ли Z1/Z2 старого калькулятора с текущими правилами CAMduct.
Подтвердить, как торцевые соединения C1/C2 влияют на PRM-001: только спецификация/BOM или площадь развёртки.
Следующее действие: перенести формулу PRM-001 в расчётный engine и покрыть тестами после финального утверждения структуры прямоугольных изделий.
```

# PRM-002 — Отвод прямоугольный

**Категория:** прямоугольные изделия
**Технический ключ старого атласа:** `rectangular-elbow`

## Назначение

```text
PRM-002 — прямоугольный отвод.
Это фасонное прямоугольное изделие для изменения направления прямоугольного воздуховода.
```

Не смешивать:

```text
PRM-002 — отвод прямоугольный
PRM-003 — колено прямоугольное переходное
PRM-004 — переход прямоугольник-прямоугольник
PRM-007 — утка прямоугольная
```

PRM-002 не должен считаться простой прямой коробкой PRM-001 без учёта дуги, угла или сегментов.

## Параметры

```text
W / A — ширина прямоугольного воздуховода, мм
H / B — высота прямоугольного воздуховода, мм
α — угол отвода, градусы
αрад = α × π / 180 — угол отвода в радианах
Rвнутр — внутренний радиус отвода, мм
Rнаруж — наружный радиус отвода, мм
Rср = (Rвнутр + Rнаруж) / 2 — средний радиус, мм
Lдуги = αрад × Rср — длина дуги по средней линии, мм
Nсегментов — количество сегментов, если отвод сегментный
s — толщина металла, мм
Z1/Z2, S1/S2, C1/C2 — замки, швы, торцевые соединения и припуски только после CAMduct/методички
```

## Sчистовая

Если отвод считается по средней длине дуги:

```text
Lдуги = αрад × Rср
αрад = α × π / 180
Rср = (Rвнутр + Rнаруж) / 2

Sчистовая = 2 × (W + H) × Lдуги / 1 000 000
```

Если отвод строится как сектор с внутренней и наружной дугой:

```text
Sвнутренняя_полоса = H × αрад × Rвнутр
Sнаружная_полоса = H × αрад × Rнаруж
Sбоковые_щёки = αрад × (Rнаруж² − Rвнутр²)

Sчистовая =
H × αрад × Rвнутр
+ H × αрад × Rнаруж
+ αрад × (Rнаруж² − Rвнутр²)
```

Если `Rнаруж − Rвнутр = W`, секторная формула сводится к:

```text
Sчистовая = 2 × (W + H) × Lдуги
```

Если отвод сегментный:

```text
Sчистовая = сумма площадей сегментов
```

Сегментную формулу не выводить произвольно без CAMduct/методички.

## Sполная

```text
Sполная = Sчистовая + технологические припуски / замки / швы / торцевые соединения
```

`Z1/Z2` из PRM-001 могут относиться к прямым/развёрточным участкам прямоугольных деталей, но для PRM-002 их нельзя переносить автоматически.  
`C1/C2` не включать в площадь без подтверждения: это может быть фланец/рейка/шинорейка как спецификация/BOM или добавка к развёртке.

Если формулы заданы в мм²:

```text
Sм² = Sмм² / 1 000 000
```

## Статус

```text
Условная — прямой проверенный CAMduct/старый калькулятор по PRM-002 в текущем реестре не найден.
Нужен вывод формулы: есть инженерная геометрия через среднюю дугу и внешний справочный источник СПВА.
Не ставить “Проверена” только из-за СПВА и не считать PRM-002 прямым воздуховодом PRM-001.
```

## Источник

```text
Геометрия прямоугольного отвода + альтернативный источник СПВА.
PRM-001 используется только как родственная прямоугольная база; Z1/Z2 и торцевые соединения не переносятся автоматически.
Прямой CAMduct/Excel/HTML-источник по прямоугольному отводу нужно подтвердить отдельно.
```

## Открытые вопросы

```text
Подтвердить по CAMduct, PRM-002 считается как гладкий радиусный отвод или сегментный отвод.
Подтвердить, как задаются Rвнутр/Rнаруж/Rср и Lдуги.
Подтвердить, применяются ли Z1/Z2 или другие замки/припуски.
Подтвердить, как C1/C2 влияют на площадь или только на BOM.
Следующее действие: подтвердить по CAMduct: радиус/угол, гладкий или сегментный отвод, Lдуги, швы/замки, торцевые соединения и припуски; затем покрыть тестами.
```

# PRM-003 — Колено прямоугольное переходное

**Категория:** прямоугольные изделия  
**Технический ключ старого атласа:** `rectangular-elbow-transition`

## Назначение

```text
PRM-003 — колено прямоугольное переходное.
Это переходное колено / отвод с переходом.
Изделие одновременно меняет направление и может менять размеры сечения.
```

Не смешивать:

```text
PRM-001 — прямой прямоугольный воздуховод
PRM-002 — отвод прямоугольный: меняет направление, сечение обычно сохраняется
PRM-004 — переход прямоугольник-прямоугольник: меняет сечение, направление обычно не меняется
PRM-007 — утка прямоугольная: смещает ось без логики переходного колена
```

PRM-003 нельзя считать простым PRM-002 или простым PRM-004 без предупреждения и CAMduct-проверки.

## Параметры

```text
W1 / A1 — ширина входного сечения, мм
H1 / B1 — высота входного сечения, мм
W2 / A2 — ширина выходного сечения, мм
H2 / B2 — высота выходного сечения, мм
α — угол колена, градусы
αрад = α × π / 180 — угол колена в радианах
Rвнутр — внутренний радиус, мм
Rнаруж — наружный радиус, мм
Rср — средний радиус, мм
Lдуги = αрад × Rср — длина дуги по средней линии, мм
Lперехода / Lосевая — длина переходной части, если она задана CAMduct
e — смещение осей, если переходное колено имеет смещение
Nсегментов — количество сегментов, если колено сегментное
s — толщина металла, мм
Z1/Z2, S1/S2, C1/C2 — замки, швы, торцевые соединения и припуски только после CAMduct/методички
```

## Sчистовая

Основная безопасная форма для панельной развёртки:

```text
Sчистовая = Sпанель_верх + Sпанель_низ + Sпанель_бок1 + Sпанель_бок2

Sпанель_i = средняя_ширина_панели_i × эффективная_длина_панели_i
```

Эффективная длина панели может быть:

```text
длина дуги;
косая / переходная длина;
длина сегмента;
CAMduct-развёртка панели.
```

Не выводить окончательные панельные формулы без подтверждения CAMduct.

Если изделие приближённо считается как “отвод + переход”:

```text
Sчистовая ≈ Sотвод_PRM-002 + Sпереход_PRM-004 − корректировка_общей_зоны
```

Простую сумму не записывать как основную формулу без предупреждения: можно задвоить общие участки или неправильно учесть переходную часть.

Если изделие считается как переходный отвод по средней линии:

```text
αрад = α × π / 180
Lдуги = αрад × Rср
Wср = (W1 + W2) / 2
Hср = (H1 + H2) / 2

Sчистовая ≈ 2 × (Wср + Hср) × Lдуги / 1 000 000
```

Это только инженерное приближение, не проверенная CAMduct-формула.

Если размеры меняются по дуге:

```text
Sчистовая = ∫ P(φ) × Rэфф(φ) dφ
```

или:

```text
Sчистовая = сумма панелей с изменяемой шириной
```

Не считать интегральную/панельную формулу проверенной без источника.

Если колено сегментное:

```text
Sчистовая = сумма площадей сегментов / панелей
```

## Sполная

```text
Sполная = Sчистовая + технологические припуски / замки / швы / торцевые соединения
```

`Z1/Z2` из PRM-001 могут применяться к прямоугольным развёрткам, но для PRM-003 требуется подтверждение CAMduct/методички.  
`C1/C2` не включать в площадь без подтверждения: это может быть фланец/рейка/шинорейка как спецификация/BOM или добавка к развёртке.

Если формулы заданы в мм²:

```text
Sм² = Sмм² / 1 000 000
```

```text

Нужен вывод формулы: конструкция понятна как переходное колено, но нельзя копировать PRM-002 или PRM-004 без CAMduct-проверки.
Не ставить “Проверена” без прямого источника CAMduct/Excel/старого калькулятора.

```

## Источник

```text

PRM-002 используется только как родственная логика изменения направления.
PRM-004 используется только как родственная логика изменения сечения.
Прямой CAMduct/Excel/HTML-источник по переходному прямоугольному колену нужно подтвердить отдельно.

```

## Параметры

```text
A1 / W1 — ширина входного сечения, мм
B1 / H1 — высота входного сечения, мм
A2 / W2 — ширина выходного сечения, мм
B2 / H2 — высота выходного сечения, мм
L / E — длина чистовой части перехода, если она задана CAMduct
D_total — полная косая длина, если используется в источнике
L_calc — расчётная чистовая косая длина с учётом прямых участков
C_axis / e — смещение по осям, мм
F/G — удлинения или прямые участки, если подтверждены

s — толщина металла, мм
Z1/Z2, S1/S2, C1/C2 — замки, швы, торцевые соединения и припуски только после CAMduct/методички
```

## Sчистовая

Основная безопасная форма:

```text
Sчистовая =
Sпанель_верх
+ Sпанель_низ
+ Sпанель_левая
+ Sпанель_правая

Sпанель_i = средняя_ширина_панели_i × эффективная_длина_панели_i
```

где:

```text
средняя_ширина_панели_i зависит от размеров соответствующих сторон входного и выходного сечения;
эффективная_длина_панели_i зависит от длины перехода, смещения и выбранной геометрии CAMduct;
при смещениях разные панели могут иметь разные косые длины;
при совпадении одной стороны количество активных соединений/панелей может отличаться.
```

Не выводить окончательные универсальные формулы панелей как “Проверена”, если нет полного подтверждения CAMduct.

## Центральный переход без смещения

Упрощённая инженерная геометрия:

```text
Sверх/низ ≈ ((A1 + A2) / 2) × Lверх/низ
Sбок ≈ ((B1 + B2) / 2) × Lбок

Lверх/низ = √(L² + ((B1 − B2) / 2)²)
Lбок = √(L² + ((A1 − A2) / 2)²)

Sчистовая ≈
2 × ((A1 + A2) / 2) × Lверх/низ
+ 2 × ((B1 + B2) / 2) × Lбок
```

Это инженерная геометрия, а не проверенная универсальная CAMduct-формула.

## Переход со смещением

```text
смещение по ширине и/или высоте меняет эффективные длины отдельных панелей;
нельзя использовать одну общую длину для всех панелей без проверки;
если смещение задано по осям, нужно отделить его от смещения относительно верхней/нижней/боковой плоскости.
```

Если переход поджат к плоскости:

```text
смещение может задаваться как разность размеров / 2
например: смещение = (B1 − B2) / 2
```

Этот вариант требует CAMduct-подтверждения.

## Sполная

```text
Sполная = Sчистовая + технологические припуски / замки / швы / торцевые соединения
```

`Z1/Z2` из PRM-001 могут применяться к прямоугольным развёрткам, но для PRM-004 требуется подтверждение CAMduct/методички.  
`C1/C2` не включать в площадь без подтверждения: это может быть фланец/рейка/шинорейка как спецификация/BOM, добавка к развёртке или часть сборочной длины.

# KRG-006 — Тройник нестандартный круглый

**Категория:** круглые изделия  
**Технический ключ старого атласа:** `round-tee-custom`

## Параметры

```text
Dст — диаметр ствола, мм
Dст1/Dст2 — диаметры ствола при переходном исполнении, мм
d — диаметр врезки, мм
L1 — длина ствола, мм
L2 — длина врезки, мм
α — угол врезки, градусы
e — смещение / нестандартное положение врезки, если задано
z — припуск на шовную сварку, мм
q — припуск на отбортовку, мм
S1/S2 — технологические припуски ствола и врезки
```

## Sчистовая

```text
Sчистовая = Sствол − Sотверстие_ствол + Sврезка_седловая

Sствол = π × D × L1

Sотверстие_ствол —
площадь области на цилиндрической развёртке ствола, которая попадает внутрь цилиндра врезки.

Sврезка_седловая —
площадь врезки по седловой развёртке, то есть по переменной длине Lэфф(φ),
полученной из линии пересечения двух цилиндров.
```

Нестандартность задаётся параметрами: углом, смещением, диаметрами, длинами и возможным переходным стволом.

```text
Тип 1: диаметр врезки может быть больше диаметра ствола.
Тип 2: ствол может иметь разные диаметры и изготавливаться как круглый переход с круглой врезкой.
```

## Sполная

```text
Sполная = Sчистовая
  + припуски S1/S2
  + отбортовка q
  + технологические добавки

Для переходного ствола дополнительно учитывается геометрия KRG-003/KRG-004.
```

## Статус

```text
Проверена по аналогии с KRG-005: используется та же геометрия пересечения цилиндров.
Отличие KRG-006 — нестандартность параметров: угол, смещение, диаметры, длины и/или переходный ствол.
Следующее действие: внедрить расчёт нестандартного пересечения цилиндров в engine и покрыть тестами по Excel-базе CAMduct.
```

## Источник

```text
KRG-005 + методичка тройника круглого + Excel-база CAMduct.

Методичка основная 30.09.2021:
раздел “32. Тройник нестандартный круглый”.

Методичка: технологическая карта идентична KRG-005; для ствола с разными диаметрами используется переход круг-в-круг.

Excel-база тройника круглого по данным CAMduct используется как подтверждение базовой геометрии KRG-005.
```

## Примечания

```text
KRG-006 использует ту же базовую геометрию пересечения цилиндров, что KRG-005; нестандартность задаётся параметрами угла, положения, диаметров и длин.
Сверка нужна на этапе внедрения engine и тестов, а не как сомнение в источнике.
```

# KRG-007 — Заглушка круглая

**Категория:** круглые изделия  
**Технический ключ старого атласа:** `round-cap`

## Параметры

```text
D — диаметр заглушки, мм
H — высота обечайки / бурта, если применяется, мм
S1FA, S1MA — припуски шва обечайки, если применяются
q / C — торцовый фальц, отбортовка или припуск соединения, если применяется
тип соединения — ниппельное / муфтовое исполнение
```

## Sчистовая

```text
Sчистовая = Sдиск + Sобечайка

Sдиск = π × D² / 4

Sобечайка = π × D × H,
если у заглушки есть цилиндрическая обечайка / бурт

Если источник подтвердит плоскую заглушку без обечайки:
Sчистовая = π × D² / 4
```

## Sполная

```text
Sполная = Sчистовая
  + технологические припуски / отбортовка / соединение,
если они подтверждены источником
```

Не добавлять S1 как продольный замок без подтверждения для конкретной обечайки заглушки.  
Не добавлять бурт H, если CAMduct/источник покажет плоский диск без обечайки.

## Статус

```text
Условная — прямой CAMduct/Excel-формулы в найденных источниках пока нет.
Методичка подтверждает конструкцию из двух деталей: обечайка и круглое основание.
Геометрическая структура требует подтверждения размеров H, припусков и влияния типа соединения.
```

## Источник

```text
Геометрия круглой заглушки + общий каталог изделий.

Методичка основная 30.09.2021:
раздел “12. Заглушка круглая”.

Методичка:
заглушка состоит из обечайки и круглого основания.

Методичка:
минимальный диаметр под ниппельное соединение — минус 3 мм от номинала;
максимальный диаметр под муфтовое соединение — плюс 6 мм от номинала.

Методичка:
обечайку сваривают по шву внахлёст S1(F)/S1(M) по 4 мм;
на обечайке выполняется торцовый поперечный одинарный фальц.
```

## Примечания

```text
Не считать формулу проверенной до подтверждения CAMduct/Excel.
Подтвердить, считается ли KRG-007 как плоский диск, диск + обечайка или диск + обечайка + технологические припуски.
Следующее действие: подтвердить наличие бурта/отбортовки по CAMduct и покрыть тестами.
```

# KRG-008 — Врезка круглая

**Категория:** круглые изделия  
**Технический ключ старого атласа:** `round-inset`

## Параметры

```text
d — диаметр врезки, мм
H — высота / длина патрубка, мм; стандартно H = 100 мм, если CAMduct/источник не задаёт другое значение
q = 10 мм — отбортовка, если выполняется из патрубка
bкольца — ширина борта отдельного кольца под отбортовку, мм; минимум 15 мм
Dвнутр = d — внутренний диаметр кольца
Dнаруж = d + 2 × bкольца — наружный диаметр кольца
A, B — размеры прямоугольной пластины, если врезка устанавливается на пластине
α — угол между осью врезки и плоскостью пластины; при α = 90° отверстие круглое, при α < 90° овальное
```

## Вариант 1 — цельный патрубок на сварке

```text
Sпатрубок = π × d × H

Если отбортовка q входит в развёртку патрубка:
Sполная = π × d × (H + q) + технологические припуски
```

Для круглого воздуховода до 450 мм используется сварка.  
Отбортовка q = 10 мм может выполняться из этого же патрубка для крепления к основному изделию.

## Вариант 2 — патрубок с отдельным кольцом под отбортовку

```text
Dвнутр = d
Dнаруж = d + 2 × bкольца

Sкольцо = π × (Dнаруж² − Dвнутр²) / 4
Sкольцо = π × ((d + 2 × bкольца)² − d²) / 4

Sчистовая = Sпатрубок + Sкольцо
```

`bкольца` задаётся клиентом/пользователем. Минимум 15 мм; типовые значения 20–30 мм не являются верхним пределом.

## Вариант 3 — патрубок на прямоугольной пластине

```text
Sпластина = A × B
Sотверстие = π × d² / 4
Sпатрубок = π × d × H

Если отверстие вырезается из пластины:
Sчистовая = Sпатрубок + Sпластина − Sотверстие
```

Если CAMduct считает пластину без вычитания отверстия, это остаётся открытым вопросом для подтверждения.

## Вариант 4 — патрубок под углом к плоской пластине

```text
Dмал = d
Dбол = d / sin(α)

Sотверстие_овал = π × Dмал × Dбол / 4
Sотверстие_овал = π × d² / (4 × sin(α))

Если врезка ставится на прямоугольную пластину и отверстие вырезается:
Sчистовая = Sпатрубок + Sпластина − Sотверстие_овал
```

При α = 90° отверстие круглое. При α < 90° отверстие становится овальным / эллиптическим.

Если под угловую врезку требуется бурт/кольцо, кольцо тоже становится овальным:

```text
Sкольцо_овал = Sнаруж_овал − Sвнутр_овал

Внутренний овал:
Dмал_внутр = d
Dбол_внутр = d / sin(α)

Если применяется упрощение “увеличение осей на 2 × bкольца”:
Dмал_наруж = d + 2 × bкольца
Dбол_наруж = d / sin(α) + 2 × bкольца

Sкольцо_овал =
  π × Dмал_наруж × Dбол_наруж / 4
  − π × Dмал_внутр × Dбол_внутр / 4
```

Способ задания наружного овального бурта/кольца должен быть подтверждён по CAMduct: равномерный борт вокруг овала или увеличение габаритных осей овала.

## Sполная

```text
Sполная зависит от выбранного варианта:
- цельный патрубок с q;
- патрубок + отдельное кольцо;
- патрубок + прямоугольная пластина;
- угловая врезка с овальным отверстием/буртом;
- технологические припуски по CAMduct.
```

## Статус

```text
Условная — KRG-008 имеет вариантную конструкцию, которую нужно подтвердить по CAMduct.
Не использовать базовую логику пересечения двух круглых цилиндров как для KRG-005/KRG-006.
Не использовать formulaState “По аналогии с KRG-005”.
```

## Источник

```text
Методичка по круглой врезке + конструктивное уточнение владельца.

Методичка основная 30.09.2021:
раздел “14. Врезка круглая”.

Методичка:
для круглого воздуховода до 450 мм используется сварка.

Методичка:
врезка изготавливается из трубы, диаметр по умолчанию минус 2 мм от номинального.

Методичка:
врезку сваривают по шву S2(Fa)/S2(Ma), накатывают зиг и формируют отбортовку 10 мм.
```

## Примечания

```text
KRG-008 имеет варианты конструкции: цельный патрубок на сварке, патрубок с отдельным кольцом под отбортовку, патрубок на прямоугольной пластине, а также угловая врезка, где отверстие на пластине становится овальным/эллиптическим. Выбор варианта, включение отбортовки/кольца/пластины и способ расчёта овального бурта должны быть подтверждены по CAMduct.
Для угловой врезки отверстие на пластине становится овальным/эллиптическим. Способ задания наружного овального бурта/кольца должен быть подтверждён по CAMduct.
Следующее действие: подтвердить в CAMduct варианты KRG-008: цельный патрубок, кольцо под отбортовку, прямоугольная пластина, угловая врезка с овальным отверстием/буртом; затем покрыть тестами.
```

# KRG-009 — Седло

**Категория:** круглые изделия  
**Технический ключ старого атласа:** `round-saddle`

## Параметры

```text
D — диаметр основного круглого ствола, мм
R = D / 2 — радиус основного ствола, мм
d — диаметр круглой врезки, мм
r = d / 2 — радиус круглой врезки, мм
α — угол между осью врезки и осью/поверхностью ствола по определению CAMduct
e — смещение врезки, если применяется
bседла — ширина борта/основания седла вокруг отверстия, если задаётся
H — высота патрубка, если патрубок входит в состав KRG-009
```

## Sчистовая

```text
Если KRG-009 является только седловым основанием/накладкой:
Sчистовая = Sседло

Если KRG-009 включает патрубок:
Sчистовая = Sседло + Sпатрубок_седловой

Если CAMduct учитывает отверстие на стволе:
Sствол_после_выреза = Sствол − Sотверстие_ствол
```

Не добавлять `Sствол` в KRG-009, если седло является отдельной деталью, а не полным тройником.

## Sотверстие_ствол

```text
Поверхность ствола разворачивается в координаты:
x — координата вдоль оси ствола
θ — угол по окружности ствола
dS = R dθ dx

Точка поверхности ствола:
P(x, θ)

Sотверстие_ствол =
  ∬ R dθ dx
по области точек P(x, θ), которые находятся внутри цилиндра врезки.
```

`Sотверстие_ствол` — площадь области на цилиндрической поверхности ствола, попадающей внутрь цилиндра врезки.

## Sседло / Sоснование

```text
Sседло — площадь основания на поверхности круглого ствола вокруг линии пересечения ствола и врезки.
```

Если CAMduct задаёт ширину седла `bседла`, наружная граница седла строится от линии отверстия с отступом `bседла`.

Открытый вопрос для CAMduct:

```text
Как задаётся наружная граница седла:
- равномерным отступом по развёртке;
- габаритным увеличением;
- табличной CAMduct-формой;
- другим производственным правилом.
```

## Sпатрубок_седловой

```text
Если патрубок входит в KRG-009:
Sпатрубок_седловой = ∫ r × Lэфф(φ) dφ

φ — угол по окружности патрубка
Lэфф(φ) — переменная длина патрубка до седлового среза,
полученная из линии пересечения двух цилиндров.
```

Если патрубок не входит в KRG-009, не включать `Sпатрубок_седловой` в `Sчистовая`. В таком случае патрубок относится к KRG-008 или к составному изделию, если так показывает CAMduct.

## Sполная

```text
Sполная = Sчистовая
  + технологические припуски
  + фальцы / отбортовки / S1/S2,
если они подтверждены CAMduct.
```

Открытый вопрос: учитывается ли `Sотверстие_ствол` как вычитаемая площадь в KRG-009 или относится к основному стволу.

## Статус

```text
Условная — прямая CAMduct/Excel-формула по седлу в текущем реестре не найдена.
KRG-009 — отдельное седло/основание, а не полная формула тройника KRG-005.
Расчётная геометрия основана на пересечении двух круглых цилиндров: круглый ствол и круглая врезка.
```

## Источник

```text
Методичка по седлу + геометрия пересечения двух цилиндров.
Связанные уточнённые карточки: KRG-005, KRG-006 и KRG-008.
Прямой CAMduct/Excel-источник по KRG-009 нужно подтвердить отдельно.
```

## Примечания

```text
Нужно подтвердить, является ли KRG-009 только седловым основанием или седлом с патрубком.
Открытый вопрос: как CAMduct задаёт наружную границу седла — равномерным отступом по развёртке, габаритным увеличением, табличной формой или другим производственным правилом.
Если патрубок не входит в KRG-009, Sпатрубок_седловой не включается в Sчистовая и относится к KRG-008 или составному изделию.
Следующее действие: подтвердить в CAMduct состав KRG-009: только седло или седло с патрубком, способ задания наружной границы седла и припуски; затем покрыть тестами.
```

# KRG-010 — Ниппель круглый

**Категория:** круглые изделия  
**Технический ключ старого атласа:** `round-nipple`

## Назначение

```text
KRG-010 — внутренний соединительный элемент.
Ниппель вставляется внутрь одной трубы одной своей половиной,
а второй половиной вставляется внутрь другой трубы.
Трубы надеваются на ниппель до упора.
Посередине ниппеля прокатывается наружный зиг / бугорок.
Наружный зиг служит ограничителем упора между двумя трубами.
```

Не смешивать с KRG-019:

```text
KRG-010 Ниппель — вставляется внутрь трубы, зиг наружу.
KRG-019 Муфта — надевается снаружи на трубу, зиг внутрь.
```

## Параметры

```text
Dном — номинальный диаметр соединяемой круглой трубы / воздуховода, мм
dниппеля — рабочий диаметр ниппеля, который должен входить внутрь трубы, мм
Δпосадки — технологическая разница диаметра для вставки ниппеля внутрь трубы, если подтверждена CAMduct/методичкой
L — полная длина ниппеля, мм
Lлевая, Lправая — рабочие половины ниппеля по сторонам от наружного зига
зона_зига — участок наружного упора посередине ниппеля
Δзиг — добавка на формирование наружного зига, если CAMduct учитывает её в развёртке
S1 — продольный шов/замок ниппеля, только если подтверждён источником
```

## Sчистовая

```text
Sчистовая = π × dниппеля × L

Если источник считает ниппель по номинальному диаметру:
dниппеля = Dном

Если источник задаёт уменьшенный посадочный диаметр:
dниппеля = Dном − Δпосадки

L = Lлевая + Lправая + зона_зига
```

Наружный зиг расположен посередине ниппеля и служит ограничителем упора между двумя трубами.

## Sполная

```text
Если зиг не меняет площадь развёртки:
Sполная = π × dниппеля × L + подтверждённые технологические припуски

Если CAMduct учитывает зиг как добавку к развёртке:
Sполная = π × dниппеля × (L + Δзиг) + технологические припуски

Если источник подтверждает продольный шов/замок именно для ниппеля:
Sполная = (π × dниппеля + S1) × L

Если формулы заданы в мм²:
Sм² = Sмм² / 1 000 000
```

Не переносить `S1` из KRG-001 автоматически без подтверждения для KRG-010.

## Статус

```text
Условная — прямой CAMduct/Excel-формулы по круглому ниппелю в текущем реестре не найдено.
Конструкция подтверждается как короткий внутренний круглый соединительный элемент.
Не смешивать KRG-010 с KRG-019: ниппель вставляется внутрь трубы, муфта надевается снаружи.
```

## Источник

```text
Конструктивная логика круглого ниппеля + общий каталог изделий.
Связанная подсказка из карточки KRG-007:
минимальный диаметр под ниппельное соединение — минус 3 мм от номинала.
KRG-001 используется только как геометрическая база круглого цилиндра;
S1, замки и припуски не переносятся автоматически.
KRG-019 должен уточняться отдельно как наружная муфта с внутренним зигом.
```

## Примечания

```text
Ниппельное соединение: ниппель вставляется внутрь одной трубы одной своей половиной, а второй половиной внутрь другой трубы; трубы надеваются на ниппель до упора.
```

# KRG-015 — Крестовина круглая

**Категория:** круглые изделия  
**Технический ключ старого атласа:** `round-cross`

## Назначение


```text

```

Если подтверждён продольный замок/шов основного ствола:

```text

```

`S1` не переносить автоматически из KRG-001 без подтверждения источником или общей логикой круглого прямошовного корпуса.

## Отверстия на стволе

Если зоны отверстий не пересекаются:

```text
Sотверстия_ствол = Sотверстие1_ствол + Sотверстие2_ствол
```

Если зоны отверстий пересекаются/накладываются:

```text
Sотверстия_ствол = S(область_отверстие1 ∪ область_отверстие2)
```

Это открытый вопрос до подтверждения CAMduct, если такие случаи возможны.

## Врезки / ответвления

Для каждой врезки:

```text
Sврезка_i_седловая = ∫ r_i × Lэфф_i(φ) dφ

r_i = d_i / 2
φ — угол по окружности i-й врезки
Lэфф_i(φ) — переменная длина врезки до седлового среза, полученная из линии пересечения цилиндров
```

Если CAMduct считает врезки как простые цилиндрические патрубки без седловой развёртки:

```text
Sврезка_i = π × d_i × H_i
```

Этот вариант требует подтверждения CAMduct.

## Sчистовая

Базовая компонентная структура:

```text
Sчистовая = Sствол − Sотверстия_ствол + Sврезка1_седловая + Sврезка2_седловая
```


Не записывать это как основную проверенную формулу без подтверждения, потому что при простой сумме двух тройников можно дважды посчитать основной ствол.

Запрещённый упрощённый вариант:

```text
S = Sтройник1 + Sтройник2
```

без корректировки общего ствола.

## Sполная

```text
Sполная = Sчистовая + технологические припуски / швы / замки / отбортовки / соединения
```

Если подтверждены швы/замки на врезках:

```text
Sврезка_i_полная = Sврезка_i_чистовая + технологические добавки_i
```

Добавлять `S1/S2`, `C1/C2`, отбортовки, фальцы и припуски только если они подтверждены источником.

Если формулы заданы в мм²:

```text

```

## Статус

```text
Условная — прямой CAMduct/Excel-формулы по круглой крестовине в текущем реестре не найдено.

```

## Источник

```text

Связанные уточнённые карточки: KRG-005, KRG-006, KRG-008 и KRG-009.

```

```text
1. переход / диффузор;
2. цилиндр;
3. зонт;
4. кронштейны;
5. фланец.
```

Активная габаритная схема для карточки: лист “Дефлектор ЦАГИ”.  
Входной параметр: `d` — базовый диаметр подключения / фланца / диффузора.

## Габаритные коэффициенты

```text
d1 = d × 1.26
d1 — верхний диаметр перехода / диффузора

d2 = d × 1.7
d2 — диаметр зонта по активной схеме “Дефлектор ЦАГИ”

h = d
h — высота перехода / диффузора

h1 = d × 0.4
h1 — расстояние от перехода до верха зонта

h2 = d × 0.3
h2 — высота зонта

D = d × 2
D — диаметр цилиндра

H = d × 1.2
H — высота цилиндра

Hфланца = 100 мм / значение CAMduct
Hфланца — высота фланца или посадочной части; пометка 100–200 мм требует подтверждения
```

```text
или 1.5d из старого/общего листа “Дефлекторы”.
```

## Переход / диффузор

Переход — усечённый конус:

```text
dниж = d
dверх = d1 = 1.26 × d
h = d

lпереход = √(h² + ((d1 − d) / 2)²)

Sпереход = π × (d + d1) / 2 × lпереход
```

## Цилиндр

```text
D = 2 × d
H = 1.2 × d

Sцилиндр = π × D × H
```

Если подтверждён продольный шов/замок:

```text
Sцилиндр_полный = (π × D + S1) × H
```

`S1` не переносить автоматически из KRG-001 без подтверждения источником или общей логикой круглого прямошовного корпуса.

## Зонт

Зонт — конус:

```text
Dзонта = d2 = 1.7 × d
h2 = 0.3 × d
Rзонта = Dзонта / 2

lзонта = √(Rзонта² + h2²)

Sзонт = π × Rзонта × lзонта
```

Если CAMduct показывает, что зонт является не полным конусом, а усечённым конусом/колпаком с отверстием, это оставить как открытый вопрос.

## Фланец / посадочная часть

В таблице есть:

```text
Hфланца = 100 мм
пометка 100–200 мм рядом с нижней частью схемы
```

Если фланец/посадочная часть считается как цилиндрический участок:

```text
Sфланец = π × d × Hфланца
```

где `Hфланца = 100 мм` или значение CAMduct/пользователя.

Если фланец считается как отдельная спецификационная деталь и не входит в площадь дефлектора, это нужно подтвердить по CAMduct.  
Не добавлять фланец в `Sчистовая` без подтверждения.



# KRG-019 — Муфта

**Категория:** круглые изделия  
**Технический ключ старого атласа:** `round-coupling`

```text

Это не основная труба и не фланец.
```

Не смешивать с KRG-010:

```text
KRG-010 Ниппель — вставляется внутрь труб, зиг наружу.
KRG-019 Муфта — надевается снаружи на трубы, зиг / упор внутрь.
```

## Параметры

```text

Dрасч — расчётный диаметр муфты, если CAMduct использует средний диаметр по толщине

z / hзиг — центральный внутренний зиг / упор, если он влияет на развёртку
q / C — отбортовки, торцевые припуски или технологические добавки, если подтверждены источником
```

Базовая инженерная структура, если муфта — цилиндрическая обечайка:

```text
Sчистовая = π × D × L
```

где:

```text
D — расчётный диаметр муфты
L — длина муфты
```

Если источник подтверждает расчёт по среднему диаметру:

```text

Sчистовая = π × Dрасч × L
```

Центральный зиг / упор фиксируется как технологическая операция, если CAMduct не добавляет его к площади развёртки.

## Sполная

```text
Sполная = Sчистовая + технологические припуски / швы / замки / фальцы / отбортовки
```

Если подтверждён продольный замок/шов именно для муфты:

Не переносить `S1` из KRG-001 или KRG-010 автоматически без подтверждения для KRG-019.

Если центральный внутренний зиг требует дополнительной развёртки или припуска:

```text
Sполная = Sчистовая + Sзиг/припуск + подтверждённые технологические добавки
```

Если зиг является только операцией гиба без изменения площади заготовки:

```text
Sполная не увеличивается на зиг.
```

Если формулы заданы в мм²:

```text
Sм² = Sмм² / 1 000 000
```

```text

По аналогии — используется геометрия цилиндрической обечайки, но KRG-019 не считается ниппелем или основной трубой.
```


Примечание: второй вариант содержит диаметр `D` в вычитании круглого отверстия. Зафиксировано как внешний вариант “прямоугольная врезка с воротником”, без смешивания с CAMduct.

## PRM-010 — Прямоугольный воздуховод с прямоугольной врезкой / прямоугольная врезка

### Альтернативный источник: СПВА

Вариант 1 — прямая прямоугольная врезка:

```text
S = 2 × (Ш + Г) × Д × n
```

Вариант 2 — прямоугольная врезка с воротником:

```text
S = [(2 × (Ш + Г) × Д + Ш × Г − π × (D/2)²) × 1.1] × n
```

Примечание: добавлено как внешний источник для сопоставления. Не заменяет существующую логику врезок.

## PRM-012 — Зонт прямоугольный

### Альтернативный источник: СПВА

Вариант 1 — зонт островного типа:

```text
S = [(Д1 × Ш1 + Д2 × Ш2 + (Д1 + Д2)/2 × В × 2 + (Ш1 + Ш2)/2 × В × 2) × 1.1] × n

Д1, Д2, Ш1, Ш2 — размеры, м
В — высота, м
n — количество, шт
```

Вариант 2 — зонт пристенного типа:

```text
S = [(Д × Ш + (Ш + П)/2 × В × 2 + Д × В) × 1.1] × n

Д — длина, м
Ш — ширина, м
В — высота, м
П — полка, м
n — количество, шт
```

Примечание: СПВА даёт формулы для прямоугольных зонтов, не для круглого зонта KRG-012.

## KMB-001 — Переход круг-прямоугольник

### Альтернативный источник: СПВА

```text
S = [((D/2 + Ш) × Д + (D/2 + В) × Д) × 1.15] × n

D — диаметр, м
Ш — ширина, м
В — высота, м
Д — длина, м
n — количество, шт
```

Примечание: эмпирическая внешняя формула СПВА с коэффициентом `1.15`. Не заменяет CAMduct.

## KMB-003 — Тройник круг-прямоугольник / круглый тройник с прямоугольной врезкой

### Альтернативный источник: СПВА

```text
S = [(π × D × Д1 + 2(Ш + Г) × Д2 − Ш × Г + 2(Ш + Г) × 0.05) × 1.2] × n

D — диаметр, м
Ш — ширина прямоугольной врезки, м
Г — глубина прямоугольной врезки, м
Д1, Д2 — длины, м
0.05 — добавка 50 мм
n — количество, шт
```

Примечание: внешняя формула для круглого ствола с прямоугольной врезкой. Не заменяет CAMduct.

## KMB-004 — Прямоугольный тройник с круглой врезкой / прямоугольный воздуховод с круглой врезкой

### Альтернативный источник: СПВА

```text
S = [(2(Ш + В) × Д1 + π × D × Д2 − π × (D/2)² + π × D × 0.1) × 1.2] × n

Ш — ширина, м
В — высота, м
D — диаметр круглой врезки, м
Д1, Д2 — длины, м
0.1 — добавка 100 мм
n — количество, шт
```

Примечание: внешняя формула для прямоугольного ствола с круглой врезкой. Не заменяет CAMduct.
