# Calc Square CAMduct V2 Brand Brief

## Product
Calc Square CAMduct V2 is an engineering web tool for ventilation product calculation, product atlas navigation, project specification assembly, and CAMduct-oriented data checking.

## Design Goal
The interface must feel like a professional technical workspace: quiet, precise, readable, and built for repeated production work rather than marketing presentation.

## Visual Character
- Clean industrial UI.
- Light theme with calm neutral surfaces.
- Orange brand accent for primary action and focus.
- Graphite typography for engineering readability.
- Compact but comfortable density.
- Clear tables, forms, and calculation panels.
- No decorative noise, blobs, gradients-as-decoration, or generic dashboard styling.

## Audience
- Ventilation engineers.
- Estimators and production managers.
- CAMduct/service reviewers.
- Clients reviewing project specifications.

## Product Priorities
1. Fast product selection through Atlas.
2. Reliable calculator inputs and outputs.
3. Clear project specification table.
4. Role-aware visibility for guest, user, client, service, and admin.
5. Traceable engineering data without exposing service/debug details to regular users.

## Brand Notes
ST Spetsmontazh and Calc Square should coexist as a practical engineering brand pair. The UI should avoid oversized logos and should prioritize workspace clarity.
